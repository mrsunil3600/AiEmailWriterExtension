## ⚙️ How It Works

### 🔍 1. Detect
- Tracks Gmail UI using `MutationObserver`
- Finds compose toolbars in real time

### 🧠 2. Think
- Sends email content to: http://localhost:8080/v1/email

### ✍️ 3. Write
- Injects AI-generated reply into editor instantly
